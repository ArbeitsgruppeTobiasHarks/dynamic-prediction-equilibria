This folder contains:

* The network tokyo_tiny: The arcs are located in `tokyo_tiny.arcs`, the commodities with inflow rates are located
  in `tokyo_tiny.demands`. This graph was taken from OpenStreetMaps, the demands were generated randomly.
* The network Sioux-Falls in the file `SiouxFalls_net.tntp`. This is a commonly used transportation network and was
  obtained from https://github.com/bstabler/TransportationNetworks.
* A picture of the synthetic network in `sample_network.png`.
